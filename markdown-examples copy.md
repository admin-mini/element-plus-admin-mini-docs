# 必要环境

nodejs16+,推荐使用nvm来切换新老项目

[下载nvm-windows](https://github.com/coreybutler/nvm-windows/releases)

## 下载

```bash
  git clone https://github.com/zjpzjp/element-plus-admin-mini.git
```
gitee镜像
```bash
  git clone https://github.com/zjpzjp/element-plus-admin-mini.git
```

## 安装

```bash
  cd element-plus/admin-mini
  npm i  #或 yarn
  npm run dev
```
